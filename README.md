# Pre-requisites
- NextJs
- Node Package Manager


# Getting started
- Clone the repository
```
git clone  https://github.com/FrozeBytes/student-management-web.git
```
- Install dependencies
```
cd <project_name>
npm install
```
- Run the project
```
npm run dev
```
  Navigate to `http://localhost:3000`

- Test the project
```
npm run test
```
